Sobre la plataforma

Este proyecto de Trabajo de Fin de Grado propone el desarrollo de una plataforma web capaz de automatizar procesos universitarios como la generación de horarios y la asignación de aulas. El sistema utiliza algoritmos de optimización para evitar solapamientos y mejorar el uso de los recursos disponibles.

¿Por qué elegir nuestra plataforma?

Nuestra plataforma no solo simplifica la gestión académica, sino que también mejora la experiencia de estudiantes y profesores al ofrecer horarios claros y espacios adecuados para cada actividad. Los alumnos pueden acceder al portal para montar su propio horario personalizado, combinando asignaturas de distintos cursos.

Funcionalidades principales

Gestión de horarios

Visualización y organización de los horarios actuales de las titulaciones universitarias.

Control de aulas

Consulta de aulas disponibles, capacidad y optimización del uso de los espacios académicos.

Generación automática

Creación automática de horarios académicos evitando conflictos de profesores, aulas y asignaturas.

Optimización

Uso de técnicas de investigación operativa para generar soluciones eficientes.
